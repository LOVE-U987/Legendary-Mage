# TRAE 个人规则配置

## 🌐 基础设置
1. **语言要求**：中文
2. **系统环境**：Windows

## 📝 代码规范
3. **函数级注释**：生成代码时添加函数级注释
4. **类级注释**：生成代码时添加类级注释
5. **参数注释**：生成代码时添加参数注释
6. **参数保护**：我修改的参数不可自行修改
7. **代码修改说明**：修改我改过的代码前请先说明
8. **修复范围**：只修改你负责的区域

## 🤖 智能体协作规则
9. **Search Agent**：项目模块多或需搜索 API 时，可创建 Search Agent
10. **多子任务并行**：任务可拆分独立子任务时，可并行提高效率。适合：批量处理独立文件；避免：任务有依赖

## 🛠️ 开发流程
11. **需求确认**：开发 UI、事件处理等复杂功能时先确认需求
12. **咨询机制**：开发中遇到困难可随时咨询
13. **Skills 优先**：优先使用已有 Skills 文件
14. **Skills 创建**：经我同意后可创建新 Skills 文件
15. **Skills 错误**：调用出错时立即报告并暂停，经同意后再继续
16. **Java 编译检查**：开发 Java 模组时，每次完成后检查编译错误

## 🐍 工具使用
17. **Python 脚本**：可使用 Python 脚本

## 🎨 交互风格
18. **语言风格**：幽默表达，可用 emoji
19. **交互方式**：引导式提问
