# 元素标记系统 - 快速入土指南

> 📚 **简要版** - 快速了解和使用元素标记系统

***

## 🎯 系统概述

这个系统允许你**自定义法术流派造成的元素标记类型**，并支持**兼容其他模组的流派**。

### 核心功能

1. **自定义元素标记** - 通过 JSON 文件配置流派造成的元素标记
2. **外部流派兼容** - 支持 Iron's Spells 及其他模组的流派
3. **Tooltip 显示** - 在游戏中查看物品时显示元素标记信息
4. **名称覆盖** - 支持修改流派的显示名称

***

## 📝 快速开始

### 步骤 1：创建 JSON 配置文件

**文件位置**：

```
src/main/resources/data/legendarymage/custom_schools/你的配置名.json
```

**基本结构**：

```json
{
  "name": "你的学派名称",
  "color": 颜色值，
  "target_school_id": "模组 ID: 流派 ID",
  "element_mark_mapping": {
    "default": "元素类型"
  }
}
```

### 步骤 2：配置元素标记

**支持的元素类型**：

- `fire` - 火焰 🔥
- `ice` - 冰霜 ❄️
- `lightning` - 雷电 ⚡
- `poison` - 毒素 ☠️
- `holy` - 神圣 ✨
- `blood` - 血色 🩸
- `eldritch` - 邪术 🔮
- `ender` - 末影 🟣

### 步骤 3：修改流派名称（可选）

**文件**：`src/main/resources/assets/legendarymage/lang/zh_cn.json`

```json
{
  "school.模组 ID. 流派 ID": "§5 你的学派名称§r"
}
```

***

## 🎨 实用示例

### 示例 1：修改末影学派

**文件**：`ender_multi.json`

```json
{
  "name": "多重末影学派",
  "color": 13369344,
  "target_school_id": "irons_spellbooks:ender",
  "element_mark_mapping": {
    "default": "ender",
    "critical": "blood"
  }
}
```

**效果**：

- 末影学派法术 → 造成末影 + 血色标记
- 可以触发更多元素反应

### 示例 2：咒刃学派配置

**文件**：`blade_school.json`

```json
{
  "name": "咒刃学派",
  "color": 7667579,
  "target_school_id": "ess_requiem:blade",
  "element_mark_mapping": {
    "default": "ender",
    "critical": "blood"
  }
}
```

**效果**：

- 咒刃法术 → 造成末影 + 血色标记
- 支持外部模组流派

### 示例 3：修改火焰学派为毒素

**文件**：`fire_to_poison.json`

```json
{
  "name": "毒焰学派",
  "color": 16711680,
  "target_school_id": "irons_spellbooks:fire",
  "element_mark_mapping": {
    "default": "poison"
  }
}
```

**效果**：

- 火焰法术 → 造成毒素标记
- 可以触发毒火、雷毒等反应

## 🎯 Tooltip 显示

系统会自动在物品的 Tooltip 中显示元素标记信息：

```
末影法术书
稀有度：稀有

元素标记：🟣 末影 | 🩸 血色
```

**支持的物品**：

- ✅ 法术书（SpellBook）
- ✅ 卷轴（Scroll）
- ✅ 法杖（StaffItem）
- ✅ 魔法武器（MagicSword）

## ⚠️ 注意事项

### 1. target\_school\_id 格式

**正确**：

```json
"target_school_id": "irons_spellbooks:fire"
"target_school_id": "ess_requiem:blade"
```

**错误**：

```json
"target_school_id": "fire"  // ❌ 缺少命名空间
"target_school_id": "irons_spellbooks/fire"  // ❌ 应该是冒号
```

### 2. 元素类型必须正确

**支持的类型**：

```json
"fire", "ice", "lightning", "poison", 
"holy", "blood", "eldritch", "ender"
```

**错误示例**：

```json
"default": "thunder"  // ❌ 应该是 "lightning"
"default": "dark"     // ❌ 应该是 "blood" 或 "eldritch"
```

### 3. 颜色代码

**语言文件中使用**：

```json
{
  "school.irons_spellbooks.ender": "§5 多重末影学派§r"
}
```

**颜色代码**：

- `§0` - 黑色
- `§1` - 深蓝色
- `§2` - 深绿色
- `§3` - 深青色
- `§4` - 深红色
- `§5` - 紫色
- `§6` - 金色
- `§7` - 灰色
- `§8` - 深灰色
- `§9` - 蓝色
- `§a` - 绿色
- `§b` - 青色
- `§c` - 红色
- `§d` - 粉色
- `§e` - 黄色
- `§f` - 白色
- `§r` - 重置（必须有！）
