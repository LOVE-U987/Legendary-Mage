## 系统概述

### 这个系统是做什么的？

这个元素标记系统允许你**通过 JSON 文件自定义法术流派造成的元素标记类型**，并且**完全兼容其他模组的流派**。

### 核心功能

1. **自定义元素标记** - 通过 JSON 文件定义流派造成的元素标记
2. **外部流派兼容** - 支持 Iron's Spells、Iron's Spells Requiem 等模组的流派
3. **Tooltip 显示** - 在游戏中查看物品时自动显示元素标记信息
4. **名称覆盖** - 支持修改流派的显示名称
5. **元素反应触发** - 积累元素标记等级触发元素反应

### 工作原理

```
1. 游戏启动
   ↓
2. 加载 JSON 配置文件
   ↓
3. 注册流派→元素标记映射
   ↓
4. 玩家使用法术
   ↓
5. 提取物品的流派信息
   ↓
6. 查询元素标记映射
   ↓
7. 施加元素标记
   ↓
8. 检查并触发元素反应
```

## JSON 配置文件详解

### 文件位置

```
src/main/resources/data/legendarymage/custom_schools/你的配置名.json
```

### 完整结构

```json
{
  "name": "流派名称",
  "color": 颜色值，
  "description": "流派描述（可选）",
  "target_school_id": "模组 ID: 流派 ID（可选）",
  "compatible_elements": ["元素 1", "元素 2"],
  "attribute_modifiers": {
    "spell_power_bonus": 0.1,
    "magic_resist_bonus": 0.15,
    "mana_cost_reduction": 0.05,
    "cast_time_reduction": 0.05
  },
  "spell_stats": {
    "damage_multiplier": 1.1,
    "range_multiplier": 1.0,
    "duration_multiplier": 1.2,
    "cooldown_reduction": 0.05
  },
  "element_mark_mapping": {
    "default": "元素类型",
    "critical": "元素类型（可选）",
    "dot": "元素类型（可选）"
  }
}
```

### 字段详解

#### 基础字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `name` | String | ✅ | 流派显示名称 |
| `color` | Integer | ✅ | 流派颜色（十进制） |
| `description` | String | ❌ | 流派描述 |
| `target_school_id` | String | ❌ | 目标流派 ID（用于兼容其他模组） |

#### 元素标记字段

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| `compatible_elements` | Array | ❌ | 兼容的元素类型列表 |
| `element_mark_mapping` | Object | ✅ | 元素标记映射配置 |

**element_mark_mapping 子字段**：
- `default` - 默认情况（必填）
- `critical` - 暴击时（可选，需要代码支持）
- `dot` - 持续伤害时（可选，需要代码支持）

#### 属性修饰符（可选）

| 字段 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `spell_power_bonus` | Float | 法术强度加成 | `0.1` = +10% |
| `magic_resist_bonus` | Float | 魔法抗性加成 | `0.15` = +15% |
| `mana_cost_reduction` | Float | 法力消耗减免 | `0.05` = -5% |
| `cast_time_reduction` | Float | 施法时间缩减 | `0.1` = -10% |

#### 法术统计（可选）

| 字段 | 类型 | 说明 | 示例 |
|------|------|------|------|
| `damage_multiplier` | Float | 伤害倍率 | `1.2` = +20% |
| `range_multiplier` | Float | 范围倍率 | `1.15` = +15% |
| `duration_multiplier` | Float | 持续时间倍率 | `1.3` = +30% |
| `cooldown_reduction` | Float | 冷却缩减 | `0.1` = -10% |

## 外部流派兼容

### 支持的模组

#### 1. Iron's Spells 'n Spellbooks ⭐⭐⭐

**支持的流派**：
- `irons_spellbooks:fire` - 火焰学派
- `irons_spellbooks:ice` - 冰霜学派
- `irons_spellbooks:lightning` - 雷电学派
- `irons_spellbooks:holy` - 神圣学派
- `irons_spellbooks:ender` - 末影学派
- `irons_spellbooks:blood` - 血色学派
- `irons_spellbooks:evocation` - 唤魔术派
- `irons_spellbooks:nature` - 自然学派
- `irons_spellbooks:anima` - 灵魂学派
- `irons_spellbooks:eldritch` - 邪术学派

#### 2. Iron's Spells Requiem ⚔️

**支持的流派**：
- `ess_requiem:blade` - 咒刃学派

#### 3. 其他任意模组 🔮

只要模组包含流派系统，都可以兼容！

**格式**：
```json
{
  "target_school_id": "模组 ID: 流派 ID"
}
```

### 配置步骤

#### 步骤 1：确定目标流派 ID

**方法**：
1. 查看模组的文档或代码
2. 在游戏中使用命令查询（如果模组支持）
3. 查看模组的资源文件

**常见格式**：
```
模组命名空间：流派路径
```

**示例**：
- `irons_spellbooks:fire` - Iron's Spells 的火焰学派
- `ess_requiem:blade` - Requiem 的咒刃学派

#### 步骤 2：创建 JSON 配置

**示例**：为咒刃学派配置元素标记

```json
{
  "name": "咒刃学派",
  "color": 7667579,
  "target_school_id": "ess_requiem:blade",
  "element_mark_mapping": {
    "default": "ender",
    "critical": "blood"
  }
}
```

#### 步骤 3：测试配置

**启动游戏后检查日志**：

**成功的日志**：
```
[INFO] 为外部模组流派配置元素标记映射：ess_requiem:blade -> legendarymage:blade_school
[DEBUG] 注册映射：ess_requiem:blade [default] -> ender
[INFO] 法术流派 - 元素标记映射注册完成：共 1 个映射
```

**失败的日志**：
```
[ERROR] 无效的 target_school_id: invalid_mod:invalid_school (在 example.json 中)
[WARN] 未知的元素类型：infire (在 example.json 的映射中)
```

### 功能说明

系统会自动在物品的 Tooltip 中显示该流派造成的元素标记信息。

### 支持的物品类型

- ✅ 法术书（SpellBook）
- ✅ 卷轴（Scroll）
- ✅ 法杖（StaffItem）
- ✅ 魔法武器（MagicSword）

### 显示效果


## 名称覆盖

### 方法 1：语言文件覆盖（推荐）

**文件**：`src/main/resources/assets/legendarymage/lang/zh_cn.json`

```json
{
  "school.irons_spellbooks.ender": "§5 多重末影学派§r",
```

```json
{

  "element_mark_mapping": {
```

```json
{
  "school.irons_spellbooks.ender": "§5 多重末影学派§r"

```

**效果**：
- ✅ 改变元素标记

## 故障排除

### 问题 1：配置不生效

**检查清单**：
1. ✅ JSON 文件位置是否正确
2. ✅ `target_school_id` 格式是否正确
3. ✅ 元素类型名称是否正确
4. ✅ 目标流派是否存在（模组是否安装）
5. ✅ JSON 语法是否正确

**调试方法**：
- 查看日志是否有错误信息
- 使用命令 `/reload` 重新加载数据包
- 检查其他配置是否生效（排除系统问题）

### 问题 2：名称无法显示

**原因**：翻译键不存在

**解决方案**：
1. **方法 1**：使用 `target_school_id`（推荐）
   ```json
   {
     "name": "咒刃学派",
     "target_school_id": "ess_requiem:blade"
   }
   ```

2. **方法 2**：添加翻译键到语言文件
   ```json
   {
     "school.legendarymage.my_school": "我的学派"
   }
   ```

### 问题 3：游戏崩溃

**可能性**：
- JSON 格式错误（缺少逗号、引号等）
- 元素类型名称拼写错误
- `target_school_id` 格式错误

**解决方案**：
1. 使用 JSON 验证工具检查语法
2. 查看崩溃日志中的错误信息
3. 逐个移除配置文件，定位问题文件

### 问题 4：Tooltip 不显示

**检查项**：
1. ✅ 物品是否包含法术（SpellContainer）
2. ✅ 流派是否有元素标记映射
3. ✅ 客户端是否加载了模组
4. ✅ 日志是否有错误信息

**调试方法**：
- 查看日志中的错误信息
- 检查物品是否有法术
- 确认映射配置正确

## 高级用法

### 1. 多流派组合配置

为一个学派配置**多种元素标记**：

```json
{
  "name": "混沌学派",
  "target_school_id": "irons_spellbooks:ender",
  "compatible_elements": ["ender", "blood", "eldritch", "poison"],
  "element_mark_mapping": {
    "default": "ender",
    "critical": "blood",
    "dot": "eldritch"
  }
}
```

**效果**：
- 不同条件下造成不同的元素标记
- 可以触发多种元素反应

### 2. 属性增强配置

为学派添加**属性加成**：

```json
{
  "name": "增强火焰学派",
  "target_school_id": "irons_spellbooks:fire",
  "attribute_modifiers": {
    "spell_power_bonus": 0.2,        // +20% 法术强度
    "magic_resist_bonus": 0.15,      // +15% 魔法抗性
    "mana_cost_reduction": 0.1,      // -10% 法力消耗
    "cast_time_reduction": 0.15      // -15% 施法时间
  },
  "spell_stats": {
    "damage_multiplier": 1.3,        // +30% 伤害
    "range_multiplier": 1.2,         // +20% 范围
    "duration_multiplier": 1.5,      // +50% 持续时间
    "cooldown_reduction": 0.2        // -20% 冷却时间
  }
}
```

### 3. 跨模组反应组合

通过配置不同模组的流派，创造**新的元素反应**：

```json
// 配置 1：让咒刃造成末影标记
{
  "name": "咒刃学派",
  "target_school_id": "ess_requiem:blade",
  "element_mark_mapping": {
    "default": "ender"
  }
}

// 配置 2：让火焰学派造成毒素标记
{
  "name": "毒焰学派",
  "target_school_id": "irons_spellbooks:fire",
  "element_mark_mapping": {
    "default": "poison"
  }
}
```

**效果**：
- 玩家同时使用咒刃和火焰法术
- 敌人身上的标记：末影 + 毒素
- 触发**末影 - 毒素反应**（虚空毒药）
